How Cover Generation Works
  1. Select a book from catalog or Google Drive source.
  2. Choose prompt source (template, AI, library, or custom).
  3. Optionally enrich metadata and generate smart prompts.
  4. Review/edit prompt in the editor before generation.
  5. Select one or more models (Gemini Flash is usually cheapest).
  6. Click Generate; each variant gets a distinct style directive + seed.
  7. Review outputs, inspect prompt text, and download per variant ZIP.
